HTML: A Declarative Language

Hypertext refers to links that connect web pages to one another, either within a single website or between websites.
Markup annotates text, images, and other content for display in a Web browser. HTML markup includes special elements such as head, title, body and so on...

The reason it seems like some HTML acts a bit like CSS in terms of how it styles diff elements is because the particular browser styles our headings

Fundamentals of HTML
Tag Description
html used to create structure and content of a webpage
body visible content on a page
p span div text blocks, span is like div but span is inline and div is block level
em strong em is kinda italic, strong is kinda bold
br ol ul li ul is bulleted, ol numbered
Tables
Tag Description
table creates a table
tr adds a row to a table
td adds data to a row
th table headings clarify meaning of data
colspan rowspan table data can span either rows or columns
thead tbody tfoot table's head body and footer

Semantic means related to the meaningt, a semantic tag hence provides context as to what the content is inside of the tag

Semantic HTML
Tag Description
header nav main footer define basic structure of web page
section defines elements with the same theme
article content that makes sense on its own: articles, blogs, comments..
aside additional related info, not req to make sense of doc..
figure figurecaption encapsulates all type of media, figcap describes media
video audio embed all media, but embed is deprecated
